## 👨‍💻 Sobre mim

Sou Técnico de TI em transição para desenvolvimento frontend, com foco em criar interfaces bonitas e funcionais. Estou construindo minha base através da trilha completa da Origamid — HTML, CSS, JavaScript, UI Design e React.
